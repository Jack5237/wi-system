# Example: Pasta & Web Development Wiki

This is a sample wiki showing how sources become interconnected knowledge.

## Starting points

- `wiki/Pasta.md` — Overview of pasta types and cooking
- `wiki/Svelte.md` — Web development framework
- `wiki/Building a Pasta E-Commerce Site.md` — Synthesis combining both topics

## How it grew

1. First source: Italian pasta types article → created Pasta, Spaghetti, Penne, Ravioli pages
2. Second source: Svelte tutorial → created Svelte, Reactivity, Components pages
3. Query: "How do I sell pasta online?" → AI synthesized a new page combining knowledge
