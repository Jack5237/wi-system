# Podcast Notes: "Building Visual Tools"

Episode featuring an xyflow maintainer discussing why they built React Flow as a headless-ish library (you bring your own styling and most of your own UI chrome) rather than a full drag-and-drop app builder. Notable quote: "We wanted to be the canvas primitive, not the product."

Also discussed: the decision to ship a separate Svelte Flow package rather than trying to make one core library framework-agnostic.
