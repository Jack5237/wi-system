# React Flow Review

React Flow is a node-based editor library for React with built-in dragging, zooming, and multi-selection. MIT-licensed, 7.4M weekly installs, used by Stripe and Typeform for workflow builders and data visualization. Built and maintained by xyflow, a small team that also ships a Svelte port.

Key strengths: minimal setup for a working node canvas, controlled/uncontrolled state modes, and a plugin-friendly architecture (custom node types, custom edges, background variants).
