# Security Policy

## Supported versions

This project currently supports the latest main branch.

## Reporting a vulnerability

Please report vulnerabilities privately to the maintainers before public disclosure.

Include:

- affected version/commit
- reproduction steps
- impact assessment
- any suggested mitigation

We will acknowledge receipt and provide a remediation timeline.

## Security notes

- Do not commit LLM API keys.
- Use environment variables for credentials.
- Review generated wiki output before publishing externally.
